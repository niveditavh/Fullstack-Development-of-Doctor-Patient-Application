# Fullstack-Development-of-Doctor-Patient-Application
Built a Doctor Patient Application . Using Java SpringBoot and React Js
Dockerised all the services.
DB used in this application is MySQL.
In this application Created - Entity, Controller,Repository and Services class
Front end technology used is Reactjs

To Run an Application use below steps-

Backend:
mvn spring-boot:run

Frontend:
cd frontend
npm install
npm start

Docker:
mvn clean package
docker compose up --build
